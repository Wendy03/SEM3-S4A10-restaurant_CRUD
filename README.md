# 餐廳清單

##### 使用者可以建立屬於自己餐廳清單，可以新增、修改、刪除餐廳清單

## Installing

#### 環境

```
- node.js v-10.15.0
- nodemon
- Express
- Express-Handlebars
- body-parse
- Mongoose

```

##### 確認本機是否安裝 Mongodb

##### 1.開啟終端機到存放專案本機位置並執行:

`git clone https://github.com/Wendy03/SEM3-S4A10-restaurant_CRUD.git`

##### 2.初始

```
1.切換目錄到專案: cd SEM3-S4A10-restaurant_CRUD
2.安裝套件: npm install
3.終端機上執行 todoSeeder.js
 - cd ~/todo/models/seeds
 - node todoSeeder.js
 - 確認Robo 3T 資料已經建立了
```

##### 3.開啟程式

`nodemon app.js`

## 主要功能

##### 1. 使用者可以新增一家餐廳

##### 2. 使用者可以瀏覽一家餐廳的詳細資訊

##### 3. 使用者可以瀏覽全部所有餐廳

##### 4. 使用者可以修改一家餐廳的資訊

##### 5. 使用者可以刪除一家餐廳
